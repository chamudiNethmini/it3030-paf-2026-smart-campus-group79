## 🧪 **QUICK TEST COMMANDS**

### Create a Ticket
```bash
curl -X POST http://localhost:8081/api/tickets \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Lab projector broken",
    "description": "Projector in Lab 1 not responding",
    "resourceId": 1,
    "priority": "HIGH"
  }'
```

### Get My Notifications
```bash
curl http://localhost:8081/api/notifications/user \
  -H "Authorization: Bearer YOUR_TOKEN"
```

### Assign Ticket to Technician
```bash
curl -X PUT "http://localhost:8081/api/tickets/1/assign?technicianEmail=tech@campus.edu"
```

### Update Ticket Status (Resolved)
```bash
curl -X PUT "http://localhost:8081/api/tickets/1/status?status=RESOLVED&resolution=Fixed%20projector%20connection"
```

### Approve a Booking (sends notification)
```bash
curl -X PATCH http://localhost:8081/api/bookings/1/status \
  -H "Content-Type: application/json" \
  -d '{
    "status": "APPROVED"
  }'
```

## 📚 **DATABASE QUERIES THAT NOW WORK**

```sql
-- Find all unread notifications for a user
SELECT * FROM notification 
WHERE user_id = ? AND is_read = false 
ORDER BY created_at DESC;

-- Find all notifications about a booking
SELECT * FROM notification 
WHERE booking_id = ? 
ORDER BY created_at DESC;

-- Find all notifications about a ticket
SELECT * FROM notification 
WHERE ticket_id = ? 
ORDER BY created_at DESC;

-- Find all TICKET_RESOLVED notifications
SELECT * FROM notification 
WHERE type = 'TICKET_RESOLVED';

-- Get unread count for user
SELECT COUNT(*) FROM notification 
WHERE user_id = ? AND is_read = false;
```
